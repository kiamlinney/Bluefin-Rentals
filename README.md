Custom car rental platform built for BlueFin Rentals LLC to migrate business operations from Turo to a self-hosted solution, 
reducing third-party commission fees and increasing brand autonomy.
- Built using React, Typescript, Tailwind CSS, and TanStack Router.
- Leveraged Supabase for database management and user authentication, ensuring secure and scalable data handling.
- Integrated Stripe to handle end-to-end payment processing.
